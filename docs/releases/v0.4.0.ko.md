# spec2openapi v0.4.0

[English release notes](v0.4.0.md)

WSDL 변환기를 크게 확장한 릴리즈입니다. substitution group을 변환부터 실제
호출까지 온전히 지원하게 됐고, WSDL 입력이 Swagger 경로만큼 유연해졌으며,
변환된 스키마에는 LLM이 인자 형식을 맞추는 데 도움이 되는 example 값이
실립니다.

## 하이라이트

**XSD substitution group을 끝까지 지원합니다.** 지금까지 substitution head
참조는 head의 base 타입으로만 변환됐습니다. 멤버 element들은 아무 기록 없이
사라졌고, head가 abstract면 어떤 값으로도 만족할 수 없는 스키마가
만들어졌습니다. 이제 head 참조는 **멤버 이름을 키로 쓰는 단일키 브랜치들의
`oneOf`**로 변환됩니다.

```json
"payment": {"oneOf": [
  {"required": ["creditCard"],   "properties": {"creditCard":   {"$ref": "…"}}},
  {"required": ["bankTransfer"], "properties": {"bankTransfer": {"$ref": "…"}}}
]}
```

JSON 값 자체가 어떤 element를 쓸지 말해 주기 때문에, SOAP 브리지는 선택된
멤버를 그대로 직렬화하고(head는 와이어에 절대 나타나지 않습니다) 응답에서는
실제로 도착한 element를 같은 형태로 되감아 줍니다. 멤버의 멤버(추이적
멤버십)와 abstract 제외까지 처리하며, 새 `x-soap-substitution` 마커가 어떤
런타임에서도 쓸 수 있게 이 계약을 문서화합니다. abstract head가 와이어에
나타나면 fault를 내는 목 SOAP 서버로 엔드투엔드 검증까지 마쳤습니다.

**WSDL 입력이 Swagger만큼 유연해졌습니다.** `convert_wsdl`(그리고
`parse_wsdl`)이 이제 다음 중 하나를 받습니다.

```python
spec2openapi.convert_wsdl("service.wsdl")          # 경로 / URL (기존)
spec2openapi.convert_wsdl("vendor-bundle.zip")     # zip 번들
spec2openapi.convert_wsdl(content=wsdl_bytes)      # 문서 본문 자체
spec2openapi.convert_wsdl(files={"service.wsdl": …, "types.xsd": …})
```

번들 안의 상대 `xsd:import`는 번들 내부에서 해소되고(`forbid_external=True`
상태에서도 오프라인으로 동작합니다), 멤버 이름은 zip-slip 공격을 차단하도록
검사하며, 압축 해제 총량에는 명시적인 상한이 있습니다. `x-soap.wsdl`에는
임시 경로 대신 논리 라벨(`<memory>`, `<bundle:entry>`,
`archive.zip!entry`)이 실리고, CLI에서는 `spec2openapi convert -`로 stdin을
읽을 수 있습니다.

**결정적인 example 값을 붙입니다.** 변환된 스키마에 원본에서 유도한
`example`이 실립니다. enumeration이 있으면 첫 값을, 날짜/시간/기간 format이면
표기법을 보여주는 고정 상수를 씁니다. 값을 지어내는 일은 없습니다. 평범한
문자열이나 숫자에는 example을 붙이지 않고, XSD `default`가 있는 프로퍼티는
default만 유지합니다.

**에러 계약이 더 단단해졌습니다.** WSDL 파싱 실패가 이제 전부
`ConversionError` 계약 안에 있습니다. strict 모드의 스킵과 변환 가능
오퍼레이션이 없는 경우까지 포함하고, 소스 라벨은 적당한 길이로 자르며, 경로
자리에 문서 *본문*을 넘기면 `content=`를 쓰라는 맞춤 안내가 나옵니다.

## 설치

```bash
pip install spec2openapi          # 변환기 + CLI (Python 3.10+)
pip install "spec2openapi[mcp]"   # + SOAP 브리지 (SOAP 서빙에 필수)
```
