# spec2openapi v0.4.0

[한국어 릴리즈 노트](v0.4.0.ko.md)

A WSDL-converter release: substitution groups are now fully supported end
to end, WSDL input became as flexible as the Swagger path, and converted
schemas carry example values that help LLMs format arguments correctly.

## Highlights

**XSD substitution groups, end to end.** A reference to a substitution
head used to convert to the head's base type only — the member elements
silently vanished, and an abstract head produced a schema no valid
instance could satisfy. A head reference now becomes a `oneOf` of
**self-describing single-key branches**:

```json
"payment": {"oneOf": [
  {"required": ["creditCard"],   "properties": {"creditCard":   {"$ref": "…"}}},
  {"required": ["bankTransfer"], "properties": {"bankTransfer": {"$ref": "…"}}}
]}
```

The JSON names the wire element itself, so the SOAP bridge serializes the
chosen member directly (the head never appears on the wire) and wraps the
actual element back symmetrically on responses. Transitive membership and
abstract exclusion are handled; the new `x-soap-substitution` marker
documents the contract for any runtime. Proven by an end-to-end test whose
mock SOAP server *faults if the abstract head ever reaches the wire*.

**WSDL input as flexible as Swagger input.** `convert_wsdl` (and
`parse_wsdl`) now accept, exactly one of:

```python
spec2openapi.convert_wsdl("service.wsdl")          # path / URL (as before)
spec2openapi.convert_wsdl("vendor-bundle.zip")     # zip bundle
spec2openapi.convert_wsdl(content=wsdl_bytes)      # the document itself
spec2openapi.convert_wsdl(files={"service.wsdl": …, "types.xsd": …})
```

Bundles resolve their relative `xsd:import`s internally (works offline
under `forbid_external=True`), member names are zip-slip-guarded, the
uncompressed size is capped loudly, and `x-soap.wsdl` shows a logical
label (`<memory>`, `<bundle:entry>`, `archive.zip!entry`) instead of a
temp path. The CLI reads stdin via `spec2openapi convert -`.

**Deterministic example values.** Converted schemas now carry
source-derived `example`s — the first enumeration value, or a fixed
canonical illustration for date/time/duration formats. No value is ever
synthesized: bare strings and numbers stay example-free, and a property
with an XSD `default` keeps only the default.

**Tighter error contract.** WSDL parse failures now stay inside
`ConversionError`: strict-mode skips and no-convertible-operations cases
included, source labels truncated, and passing document *content* where a
path is expected gets a targeted hint pointing at `content=`.

## Install

```bash
pip install spec2openapi          # converter + CLI (Python 3.10+)
pip install "spec2openapi[mcp]"   # + SOAP bridge (required to serve SOAP)
```
